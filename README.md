# 💻 Sistema de Agendamento de PCs

Sistema web desenvolvido em PHP para gerenciamento e agendamento de computadores.  
O objetivo do projeto é organizar o uso das máquinas, evitando conflitos de horários e facilitando o controle por parte dos usuários.

---

## 🚀 Funcionalidades

- 📅 Agendamento de horários
- 👤 Cadastro de usuários
- 🖥️ Controle de computadores disponíveis
- ✏️ Edição e cancelamento de reservas
- 🔒 (Opcional) Sistema de login/autenticação

---

## 🛠️ Tecnologias Utilizadas

- PHP
- MySQL
- HTML5
- CSS3
